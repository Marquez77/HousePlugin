# HousePlugin

The Minecraft House Plugin Commission.

# Commands
> **User Command**
* /house help
* /house invite <playername>
* /house invite accept
* /house kick <playername>
* /house info
* /house teleport
* /house doorautoclose <number>
* /house leave

> **Admin Command**
* /house create <name>
* /house delete <name>
* /house buysign <name>
* /house reload
